# LLM

## 1 大模型和大语言模型的区别
**大模型**是指参数数量巨大（通常达到数十亿、数百亿甚至万亿级别）、训练数据量庞大、需要大量计算资源的深度学习模型。“大” 主要体现在参数规模、数据量和算力需求上。应用领域不局限于某一特定领域，可以处理多种模态的数据。
+ GPT系列、PaLM（大语言模型，处理文本）
+ DALL-E、Stable Diffusion（大模型，处理文本到图像）
+ AlphaFold（大模型，处理蛋白质结构预测）
+ Whisper（大模型，处理语音识别和翻译）

**大语言模型**是大模型的一种特定类型，其训练数据主要来自文本和代码，核心能力是理解、生成和处理人类语言。专精于语言任务。
应用领域包括文本生成、对话、翻译、摘要、代码编写等所有与语言相关的任务。
+ ChatGPT、GPT-4、LLaMA、文心一言、通义千问。
